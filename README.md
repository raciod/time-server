# Time Server

A simple time server for Unix systems.

## Installation

Clone the repository:

```bash
git clone https://github.com/raciod/time-server
```

Navigate into the project directory:

```bash
cd time-server
```

Build the project:

```bash
make
```

## Usage

Once built, open your browser and visit:

```
http://127.0.0.1:8080
```
**Update: This program can now be used with both IPv4 and IPv6.**
